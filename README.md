# Negative-controls-in-MR

This repository contains R code used to implement the real-data analysis of the paper "Negative control outcomes and selection bias in Mendelian randomization". The analysis uses summary-level genetic data from [Schoeler et al. (2023)](https://www.nature.com/articles/s41562-023-01579-9). The datasets are freely available for download [here](https://drive.google.com/drive/folders/1QMq9Y_BDK-9o0TlaJL-fwaqQl4BbR6Ek).
